# Pattern-Samples
My design patterns samples on C#
## Usage
F.e. check sample of using Singleton
```c#
using System;

namespace StudyPatterns
{
    class Program
    {
        static void Main(string[] args)
        {
            SingletonTest.MainProgram();
            Console.ReadLine();
        }
    }
}
```
So just invoke (PatternName)Test.MainProgram(); in Program.cs
